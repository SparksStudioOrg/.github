# 🔐 Security Policy

The security of our projects is important to us.

## Reporting a vulnerability

If you discover a security vulnerability, **please do not open a public issue immediately**.

Instead, contact us privately:

[SparksStudioOrg@proton.me](mailto:SparksStudioOrg@proton.me)

We will investigate the issue and respond as quickly as possible.

Thank you for helping keep Sparks' Studio secure.
