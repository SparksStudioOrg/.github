# 🤝 Contributing

Thank you for your interest in contributing to **Sparks' Studio**.

We welcome contributions of all kinds:

* 🐛 Bug reports
* ✨ Feature ideas
* 🔧 Pull requests
* 📖 Documentation improvements

---

## How to contribute

1. Fork the repository
2. Create a new branch for your change
3. Make your changes
4. Commit your work
5. Open a Pull Request

Please keep pull requests small and focused so they are easier to review.

---

## Commit messages

Try to write clear commit messages. For example:

Fix crystal spawning bug
Add new player animation
Improve README documentation

---

## Developer Certificate of Origin (DCO)

By contributing to this project, you agree that your contribution is licensed under the project's license.

Please sign your commits using the **DCO sign-off**:

git commit -s -m "Your commit message"

This adds a line like:

Signed-off-by: Your Name [your@email.com](mailto:your@email.com)
